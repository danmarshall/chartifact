Welcome to Chartifact Atlas

An enhanced editor for interactive documents with live preview.

How to load a document:

  • Paste: Copy and paste your markdown content into the editor
  • Drag and drop: Drop a markdown file onto the editor pane
  • URL parameter: Add ?load=url/to/your/file.md to the page URL
  • File upload: Click the "Upload" button to choose a file

Examples:

  • Seattle Weather Example 1
  • Sales Dashboard
  • Movie Picker

Features:

  • Live Preview: See changes instantly as you type
  • Split View: Edit and preview side-by-side
  • Format Toggle: Switch between Markdown and JSON editing
  • Resizable: Drag the center divider to adjust pane sizes
