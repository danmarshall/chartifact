Chartifact Interactive Document Host

You can render interactive documents here via several methods:
  • Clipboard: paste your markdown content directly when this page has focus
  • Drag and drop: drop a markdown file onto this page
  • URL parameter: add /?load=url/to/your/file.md to the url
  • File upload: click the button below to choose a file
  • PostMessage: send markdown content to this page using the window.postMessage API with the property markdown
