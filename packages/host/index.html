Interactive Document Host

You can render interactive documents here 4 ways:
  • Via clipboard: paste your markdown content directly when this page has focus
  • Via drag and drop: drop a markdown file onto this page
  • Via url parameter: add /?load=url/to/your/file.md to the url
  • Via file upload: click the button below to choose a file
  • Via postmessage: send markdown content to this page using the window.postMessage API with the property markdown
